# Start C#

### Down load .NET
- [Download .NET for mac](https://dotnet.microsoft.com/en-us/download) 
 
 ```bash
 # test dotnet installed
    dotnet --version
# Create new project
    dotnet new --list
    dotnet new console

    dotnet build
    
    # run the app 
    dotnet run
    # output: Hello World!
 ```
